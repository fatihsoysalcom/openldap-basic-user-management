# OpenLDAP Basic User Management

This script demonstrates fundamental OpenLDAP administrative tasks using command-line tools. It shows how to add, search for, and delete users from an LDAP directory, illustrating basic directory management operations.

## Language

`bash`

## How to Run

1. Ensure OpenLDAP is installed and running.
2. Configure the BASE_DN, ROOT_DN, and ROOT_PASSWORD variables in the script.
3. Make the script executable: chmod +x manage_ldap_users.sh
4. Run the script: ./manage_ldap_users.sh
